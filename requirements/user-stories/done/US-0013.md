## User Story

As a **developer**,  
I want **to have clear, type-safe enums for TransitionStatus and ComponentLifecycleStatus**,  
so that **the component state machine is unambiguous and compiler-checked**.

- [x] New file `src/component_types.h` created with enum definitions

- [x] Corresponding implementation in `src/component_types.cpp`

- [x] Documentation update in `docs/` not required (terminology unchanged)
