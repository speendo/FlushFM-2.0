# US-0025f: Event Call-Site Migration to Generic Contract

> **Status:** Done | **Priority:** High | **Created:** 2026-04-25

## User Story
As a **maintainer**, I want **to migrate controller-facing event call sites to the agreed generic contract and Mailbox flow**, so that **the controller event surface is minimal and consistent**.

## Acceptance Criteria
- [ ] Controller-facing call sites use the agreed contract consistently and route supervisor-bound events through the single-slot Mailbox
- [ ] Legacy names remain only where adapter translation is intentional outside the supervisor event flow
- [ ] Arbitration behavior remains deterministic and unchanged

## Notes
This story depends on contract definitions from US-0025c, US-0025d, and US-0025e. It focuses on call-site migration only; member cleanup is in US-0025g.
