> **Status:** Done | **Priority:** Medium | **Created:** 2026-04-25

## User Story
As a **maintainer**, I want **to remove controller self-registration coupling and align component registry ownership**, so that **the component registry contains only runtime components**.

## Acceptance Criteria

- [x] Component lifecycle registry is limited to runtime components
- [x] Static debug label remains diagnostics-only
