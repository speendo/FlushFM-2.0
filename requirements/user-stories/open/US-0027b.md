# US-0027b: Controller-as-Module Refactor Contract

> **Status:** To Do | **Priority:** Medium | **Created:** 2026-04-25

## User Story
As a **maintainer**, I want **to remove controller self-registration coupling and align component registry ownership**, so that **the component registry contains only runtime components**.

## Acceptance Criteria
- [ ] Controller does not participate in component registry identity flow
- [ ] Component lifecycle registry is limited to runtime components
- [ ] Static debug label remains diagnostics-only

## Notes
This story depends on US-0027a and implements the module-boundary contract.
