## Notes

*This is the foundation step before all other user stories. Keep it minimal but functional.*

**Required project structure:**
```
FlushFM 2.0/
├── src/                    # Source code
│   ├── main.cpp           # Main Arduino file
│   └── config.h           # Basic configuration constants
├── lib/                   # Custom libraries (empty for now)
├── data/                  # Web UI files (empty for now)
└── platformio.ini         # Platform configuration (or Arduino IDE setup)
```

**Minimal boilerplate requirements:**
- Serial initialization (115200 baud)
- WiFi scan functionality (no connection required yet)
- Basic error handling (try/catch or status checks)
- Board identification (confirm ESP32-S3 detection)
- Memory info display (Flash, PSRAM availability)

**Libraries to include:**
- Arduino-ESP32 framework
- WiFi library (built-in)
- No external libraries yet - keep it minimal
